personal_inf_sec_bot

Функционал бота:
1. Анализ ресурсов и угроз
Проверка ссылок/сайтов: Безопасность URL (на фишинг, вредоносное ПО).
Whois-проверка: Информация о домене (дата создания, владелец).
Проверка утечек паролей (Breach Scan): Пробив email по базам утечек (например, HaveIBeenPwned).
Анализ заголовков безопасности сайта: Проверка наличия SSL, HSTS, CSP и др.

2. Криптография и сеть
Проверка SSL/TLS сертификатов: Валидность, дата expiry, issuer.
Генератор паролей: Сложные пароли заданной длины и набора символов.
Проверка надежности пароля: Оценка стойкости (энтропия).
